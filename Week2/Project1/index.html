Tags and Descriptions

Tags Inline/Block Description
<header> Block Represents a container for introductory content or a set of navigational links
<nav> Block Defines a section of navigation links
<main> Block Specifies the main content of a document
<section> Block Defines a section in a document
<article> Block Represents a self-contained composition in a document
<aside> Block Defines content aside from the main content
<footer> Block Defines a footer for a document or section
<h1> to <h6> Block Heading elements that represent headings of different levels
<p> Block Defines a paragraph
<a> Inline Creates a hyperlink to other web pages or locations
<strong> Inline Indicates strong importance or emphasis
<em> Inline Indicates emphasis that subtly changes the meaning of a sentence
<span> Inline Defines a section in a document that needs to be styled or manipulated
<img> Inline Inserts an image into the document
<ul> Block Defines an unordered list
<li> Block Defines a list item in an unordered list
